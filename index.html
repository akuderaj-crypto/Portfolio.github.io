♻️ Ai4A Project: Smart Waste Classification System

Overview: An AI-powered waste segregation system that uses a Teachable Machine image classification model inside MIT App Inventor to detect dry vs. wet waste in real time, then sends the result over Bluetooth to an Arduino for hardware-level sorting.

Tech Stack: MIT App Inventor, Teachable Machine, Arduino Uno, HC-05 Bluetooth Module, Text-to-Speech

App Inventor logic: classification & Bluetooth messaging

Live demo: app classifying waste in real time

📚 DSC Project: Library Management System

Authors: Created as a group project by Raj Akude at MIT Vishwaprayag University under the guidance of Prof. Kalicharan Sir.

Core Functions (CRUD): Supports adding new books, displaying a formatted list, searching by ID, updating existing details, and deleting records.

Smart Memory Management: Uses dynamic allocation (malloc and free) so there is no fixed limit on the number of books, and cleans up all memory automatically upon exiting to prevent leaks.
